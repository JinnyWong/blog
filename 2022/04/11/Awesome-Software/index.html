What's on my PC | 好玩好用的软件分享

发表于2022-04-11|有趣
|阅读量:

一些我觉得好用和好玩的软件记录~
顺便避免我下次重装系统又漏装一堆软件然后要急用时阵亡😅

第二个大脑

Obsidian
用来记录我的每日代办、反思、还有得到的启发。

Notion
用来做健身记录、一些学习笔记还有我的全年规划。

浏览器

Chrome
平时用的浏览器。

Tor Browser
要找软件、电影资源的torrent时用的

学习工具

腾讯会议、Zoom Meeting、DingTalk
这些就不用多说了，网课人的必备软件:)

Zotero
一个文献管理软件，可以使用插件直接将引用文献的链接加入论文里。

Microsoft Suite全家桶
其实主要也是使用 Word, Excel, Powerpoint 而已。

Drawboard PDF
一个我认为很好用的PDF Editor！主要让我喜欢的点是可以在各种电子课本之间快速切换。

XMind、draw.io
制作思维导图的软件。

编程工具

Visual Studio Code
万能武器

Typora
快速edit Markdown。

GitHub Desktop
因为我懒惰Git bash哈哈哈哈

设计工具

Adobe全家桶
主要使用 Photoshop和Lightroom。

Maya
3D模型建模软件

影视工具

ScreenToGIF
将录屏导出成GIF模式。适用于快速且简短，而且不用说话的录屏教学时。

OBS
一款强大的录制软件。可以录屏、录摄像头、录声音。

Audacity
快速录音和剪辑音频的软件，很适合英语课的oral考核使用。

DaVinci Resolve 17
功能强大的免费视频剪辑软件，简直是业界标配！对于新手来说有点难上手。

Bandlab
免费的DAW，但是我还没怎么玩过:)

Voicemeeter
调节麦克风声音大小和色调。

娱乐

Epic Games Launcher
主要是想领Epic爸爸送的免费游戏，但除了Overcooked 2剩下的其实还挺少玩。

Steam
常玩的有Apex Legends跟 CSGO。

Discord
跟宝贝朋友们嗨时用的，语音通话超方便。

Spotify
听歌啦📻

桌面美化

Custom Cursor
TranslucentTB
Rainmeter
